발행 스케줄

글 추가하는 방법
01
posts.json 파일 열기
GitHub 저장소에서 posts.json 파일을 클릭하고 우측 연필 아이콘(✏️)으로 편집 모드 진입
02
포스팅 객체 추가
맨 끝 항목 뒤에 ,{ ... } 형태로 새 포스팅 추가. id는 이전 번호 +1, status는 "draft"로 시작
03
저장 (Commit changes)
초록색 Commit changes 버튼 클릭 → 1~2분 후 이 페이지 새로고침하면 바로 반영
04
상태 변경
각 카드의 승인 버튼으로 draft → approved 변경 가능. 발행 후엔 status를 "posted"로 수정
클립보드에 복사됐어요 ✓
